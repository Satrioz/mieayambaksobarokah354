# 🍜 Mie Ayam Bakso Barokah 354 - Website Landing Page

Website landing page responsif untuk usaha kuliner **Barokah 354** yang berlokasi di Sidorejo, Lampung. Dibuat dengan fokus pada kecepatan, desain modern, dan kemudahan informasi menu bagi pelanggan.

## 🚀 Fitur Utama

* **Dynamic Pricing System**: Integrasi dengan API Hari Libur Nasional. Harga makanan otomatis naik Rp1.000 pada hari libur, sementara harga minuman tetap terkunci di Rp5.000.
* **Responsive Design**: Tampilan yang rapi di berbagai perangkat (Mobile, Tablet, Desktop) menggunakan Tailwind CSS.
* **Interactive Menu**: Daftar menu makanan dan minuman yang dilengkapi dengan deskripsi detail dan foto yang menggugah selera.
* **Testimonial Slider**: Menampilkan ulasan asli pelanggan langsung dari Google Maps secara dinamis.
* **Direct Contact**: Tombol pesan cepat yang terhubung langsung ke WhatsApp dan fitur kirim masukan via Email.

## 🛠️ Tech Stack

* **HTML5 & CSS3** (Custom Animations)
* **Tailwind CSS** (Utility-first Framework)
* **Vanilla JavaScript** (ES6+, Fetch API, Intersection Observer)
* **Hari Libur API** (Data hari libur nasional otomatis)
